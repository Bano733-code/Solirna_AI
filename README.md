# Solirna AI

Solirna AI is an autonomous AI startup co-founder that helps founders build investor-ready startups using Fireworks AI.

## Features

- AI Chat
- Startup PRD Generation
- Pitch Deck Generation
- Persistent Memory
- Semantic Search
- PostgreSQL + pgvector
- Fireworks AI LLM
- JWT Authentication

---

## Tech Stack

- FastAPI
- PostgreSQL
- pgvector
- SQLAlchemy
- Fireworks AI
- Docker

---

## Installation

Clone the repository

```bash
git clone https://github.com/Bano733-code/Solirna_AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create

```
.env
```

using

```
.env.example
```

Run

```bash
uvicorn app.main:app --reload
```

---

## Docker

docker compose up --build
